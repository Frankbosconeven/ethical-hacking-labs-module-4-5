## Overview
This repository contains documentation of practical ethical hacking labs completed as part of my cybersecurity training.  
All activities were carried out in controlled lab environments strictly for **educational and ethical purposes only**.

The labs focus on understanding how common attacks work so that systems can be better secured and defended.

---

## Labs Documented

### 1. Browser Exploitation Framework (BeEF)
- Browser hooking using BeEF demo pages
- Fake Notification Bar social engineering attack
- Tabnabbing attack
- Understanding client-side attack techniques

### 2. Website Cloning Lab
- Cloning a test website using social engineering tools
- Understanding how credential harvesting attacks work
- Awareness of phishing and cloned website threats

### 3. SMB Vulnerability Scanning (Enum4Linux)
- Scanning SMB services for misconfigurations
- Enumerating users, shares, and OS information
- Identifying exposed SMB information
